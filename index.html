Ideation Process

For this project I had decided to scrap my idea from project 1 and then create a new idea for project 2. This was because my project 1 was slightly too challenging to create a prototype for and was lacking in material. However when coming up with concepts, I did consider trying to use my project 1 material. A couple ideas that I had were as followed.

September 1st 2020

1. Using a disco ball as a "helmet" that acts as a social media platform for individual homes due to COVID-19 so that families can communicate through video chat using a 3d image.

2. Create a new phone application targeted at the older generations to teach them how to use their cellphones. This would use wireframes and create a working prototype video.

3. Using the disco balls in the first assignment to create a gallery presentation that instead of having a video about the internet playing, uses hyper reality that the user can play games with, do a quiz, take a photo, etc.

4. Create an application that uses hyper reality in retail where your phone can take a photo of any object anywhere and it will tell you how much it costs and where to buy it.

5. Go off my first initial idea of the disco ball gallery and use 3d rendering and potentially animation to create a mock-up gallery scene.

Final Decision

September 8th 2020

Through this iteration process, I have decided to go with option 2 and create a new application designed for the older generation who struggle to use their cellphones. The application will be for iphones only and contain a range of different options such as games to test your phone function knowledge, AR simulation to teach them (which would be included as part of the games), settings to adjust the phones font, sound, etc. and tutorials to teach them. I will use the program JustInMind to create a mock up interactive prototype.

This project will help start a discussion about how techology is becoming less and less accessible for those who struggle. It is becoming increasingly difficult for people to learn these new skills as new technology advances are happening daily. As a society we cannot leave these older generations without technology skills and it is our responsibility as the "tech generation" to aid and help whereever we can.

I will start my project with gathering resources such as creating personas, wireframes and getting inspiration.
